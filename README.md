# linebot
![image](https://github.com/F74064046/linebot/blob/master/fsm.png)

##目的:
無聊的時候可以用這個玩幾A幾B打發時間

##使用方法:
輸入任意文字即可開始
依照機器人所說的輸入正確input來開始遊戲

輸入start可看遊戲說明

接著輸入ok來開始遊戲

輸入猜的數字玩遊戲

機器人會回傳?A?B來提示

直到4A0B,遊戲結束

如果想再玩一次,輸入start
